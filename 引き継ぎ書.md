最終更新: 2026-03-07

## 概要

少年サッカー（8人制）の試合記録アプリ。スマホで出場選手・スコア・得点者・フォーメーションを入力し、テキスト形式で出力できる。

## 現在のバージョン: v1.4.5

### 主な機能

| 機能 | 説明 |
|------|------|
| **出場選手選択** | タップで選択/解除、ダブルタップでGK |
| **フォーメーション** | 3-2-2 / 3-3-1 / 2-3-2 から選択、配置図を出力 |
| **交代入力** | 個別交代（OUT→IN形式）または総入替え |
| **スコア・得点者** | タップで得点追加、長押しでリセット |
| **PK戦** | ○×で結果入力 |
| **リアルタイム共有** | Firebase経由で複数人で同時入力可能 |
| **テキスト出力** | 背番号版/名前版を選択してコピー |

### v1.4.5 の変更点（v1.3.0から）

1. **フォーメーション配置機能**
   - 「配置図」ボタンでモーダルを開く
   - FWが上、GKが下のレイアウト
   - ポジションをタップ → 選手をタップで配置
   - テキスト出力に配置図を含める
   - 交代がある選手は `(→XX)` を表示

2. **GK選択方法の変更**
   - 長押し → **ダブルタップ**に変更（反応が良い）

3. **出力形式の改善**
   - 出場選手の区切り: `.` → `,`
   - 交代表記: `スタメン全員→交代IN` → `OUT→IN` 形式

4. **UI改善**
   - 登録メンバー: 展開式（デフォルト非表示）
   - フォーメーション図: 中央揃え対応

## 操作方法

### 選手選択
- **タップ**: 選択/解除
- **ダブルタップ**: GKに設定
- **GKドロップダウン**: GKを直接選択

### 得点者
- **タップ**: +1点
- **長押し**: リセット

### フォーメーション
1. 「配置図」ボタンをタップ
2. フォーメーション（3-2-2など）を選択
3. ポジションをタップ → 選手をタップで配置
4. 「保存」で試合に紐付け

## 既知の課題・注意点

### 選手マスタの同期問題
- DEFAULT_ROSTERはHTMLにハードコード
- ローカルストレージに古いデータがあると、そちらが優先表示される
- 別端末で開くとDEFAULT_ROSTERに戻る
- **対策**: 選手変更時はDEFAULT_ROSTERを更新してGitHubにpush

### Firebase共有
- ルームID = 試合日（8桁）
- 同じ日のデータは上書き
- 履歴蓄積機能は未実装
